# Neue Montreal Font Host

A simple Next.js application to publicly host the Neue Montreal font family with CORS enabled.

## Fonts Included

- Neue Montreal Regular
- Neue Montreal Italic
- Neue Montreal Light
- Neue Montreal Light Italic
- Neue Montreal Medium
- Neue Montreal Medium Italic
- Neue Montreal Bold
- Neue Montreal Bold Italic

## Deployment

This project is ready to deploy to Vercel:

1. Push to GitHub
2. Import in Vercel
3. Deploy

## Usage

After deployment, fonts will be available at:
```
https://your-domain.vercel.app/fonts/NeueMontreal-Regular.otf
```

All fonts are served with CORS headers enabled for cross-origin requests.

## Local Development

```bash
npm install
npm run dev
```

Visit `http://localhost:3000` to see the font directory.
